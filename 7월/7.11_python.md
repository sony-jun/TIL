### 변수

- 수학에서 말하는 x,y

- 가변적인 정보를 저장

- EX)

  ~~~python
  a = 5
  print(a)
  # 결과값은 5
  a = '이름'
  print(a)
  # 결과값은 [이름]
  ~~~

- 코드는 위에서부터 아래로

- 코드는 오른쪽에서 왼쪽

- ex)

  ~~~python
  a = 5
  a = a + 1
  print(a)
  #결과값은 6
  ~~~

- 변수는 수가 아니라도 가능

- ex)

  ~~~python
  a = '파이썬'
  a = a * 3
  print = a
  #결과값은 "파이썬파이썬파이썬"
  ~~~
